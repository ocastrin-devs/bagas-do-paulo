O que é HTTPS?

Cliente

O cliente é qualquer dispositivo ou programa (como o seu navegador Chrome ou Edge) que solicita dados a um servidor através de uma rede. Ele é a ponta que "pede" a informação.

Servidor

O servidor é um computador robusto que armazena os arquivos do site e "serve" essas informações quando o cliente faz um pedido. Ele fica aguardando requisições 24h por dia.
